# 国土交通省 川の防災情報（水害・土砂災害）

出典: https://www.mlit.go.jp/river/bousai/bousai-portal/index.html

## 主要な情報源

- **国土交通省「川の防災情報」**: 気象・河川・土砂災害等の情報を1画面でまとめて確認できる。
- **国土交通省ハザードマップポータルサイト**: 全国の市町村のハザードマップや災害リスク情報を提供。

## 災害時に活用できるコンテンツ

- **気象状況**: 雨量や河川の水位などを確認できる。
- **被害想定**: 洪水、内水氾濫、土砂災害、高潮などの被害状況やハザードマップを確認できる。

## 学習ツール

- **マイ・タイムライン**: 水害対策として、いざというときに1人1人がすべき行動とそのタイミングを学べるクイズ形式の教材。

これらのリソースにより、河川洪水や土砂災害に関する予防対策と緊急時対応の情報が統合的に提供されている。
