# 東京都防災ホームページ

出典: https://www.bousai.metro.tokyo.lg.jp/

## 地震対策

- 感震ブレーカー（「グラぴたスイッチ」等）の設置により、地震による火災や延焼を防止することを推奨している。
- 令和4年5月25日公表の「首都直下地震等による東京の被害想定」を参考資料として提供している。

## 風水害対策

- 台風や集中豪雨への対応ガイドを提供している。
- 降雨情報システム「東京アメッシュ」でリアルタイムの降水量を確認できる。
- 気象庁の警報・注意報、台風情報へのリンクを掲載している。

## 避難情報・施設

- 帰宅困難者対策として「キタコンDX」システムを導入し、GPSを活用して一時滞在施設の情報をLINEで配信している。
- マンション防災として、約900万人の住民向けに「在宅避難」の備えを強調している。
- 区市町村の防災担当窓口一覧を掲載している。

## 備え・啓発

- 防災ブック「東京くらし防災」「東京防災」（令和5年9月リニューアル版）を提供している。
- 東京都防災アプリでコンテンツを利用できる。
- 毎月1日・15日は「災害伝言サービス体験日」としている。
