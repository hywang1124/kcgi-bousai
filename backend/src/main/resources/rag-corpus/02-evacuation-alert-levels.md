# 避難はいつ、どこに：避難情報・警戒レベル・屋内安全確保

## 避難情報の受け取り方

市町村が避難情報を発令した場合、テレビ、ラジオ、インターネット、自治体ホームページ、防災アプリ、防災行政無線、広報車、緊急速報メールなどで伝えられます。災害時は一つの情報源だけに頼らず、複数の方法で確認します。停電や通信障害に備え、ラジオ、モバイルバッテリー、紙のハザードマップも用意しておきます。

避難情報は、必ず警戒レベル1から5まで順番に出るとは限りません。状況が急に悪化し、市町村から避難情報が出る前に危険が迫る場合もあります。気象庁の防災気象情報、河川水位、雨量、土砂キキクル、洪水キキクル、自治体情報を参考にし、自分や家族の状況に合わせて早めに判断します。

## 警戒レベルと基本行動

警戒レベル3「高齢者等避難」が出たら、高齢者、障害のある人、乳幼児、妊婦、外国語で情報を得にくい人、避難に時間がかかる人とその支援者は危険な場所から避難を開始します。それ以外の人も、避難準備や自主避難を検討します。

警戒レベル4「避難指示」が出たら、危険な場所にいる人は全員避難します。洪水、土砂災害、高潮、津波などの危険区域にいる場合は、避難場所、安全な親戚・知人宅、ホテル、頑丈な建物の上階など、状況に応じて命を守れる場所へ移動します。

警戒レベル5「緊急安全確保」は、すでに災害が発生または切迫し、通常の避難が危険な段階です。この段階を待ってはいけません。もし外へ避難することがかえって危険な場合は、自宅や近くの建物の上階、崖から離れた部屋、より高い場所など、少しでも命が助かる可能性が高い場所へ移動します。

## どこへ避難するか

避難は「避難所へ行くこと」だけではありません。立退き避難が必要な場合は、市町村が指定した災害種別に対応する指定緊急避難場所、安全な親戚・知人宅、ホテル、職場などへ移動します。洪水などで自宅にとどまることが可能かどうかは、ハザードマップで家屋倒壊等氾濫想定区域に入っていないこと、浸水深より高い階に安全な部屋があること、水が引くまで過ごせる備蓄があることなどを確認して判断します。

自宅にとどまる「屋内安全確保」は、外へ移動するより安全な場合に限られます。浸水が深くなる地域、家屋倒壊のおそれがある区域、土砂災害警戒区域、津波浸水想定区域では、屋内にいるだけでは危険な場合があります。災害種別ごとに、どの場所が安全かを事前に確認します。

## 避難時の注意

避難は、暗くなる前、風雨が強くなる前、道路が冠水する前に行います。川、用水路、海岸、地下道、アンダーパス、崖、斜面、壊れそうな橋には近づきません。車での避難は渋滞、浸水、道路損壊で動けなくなることがあるため、自治体の指示や地域の状況に従います。

持ち物は最小限にして両手を空けます。長靴は水が入ると歩きにくいため、歩きやすい靴を選びます。夜間は懐中電灯を使い、複数人で声を掛け合って移動します。避難の途中で忘れ物に気づいても、危険な場所に戻ってはいけません。

## 情報がない・判断に迷う場合

避難場所の開設状況、道路の通行可否、地域ごとの避難指示、避難所の混雑、福祉避難所の利用、ペット同行避難などは、自治体ごとに異なります。資料にない個別の場所、電話番号、現在の開設状況は推測せず、自治体の公式ホームページ、防災アプリ、コールセンター、消防・警察などの公式情報を確認します。
